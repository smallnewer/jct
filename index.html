JCT

==> javascript : css3 transform/transition

一款方便js操作css3 transform、transition的工具. 正在开发。 框架整体结构类似jquery，能够快速上手。全局就一个对象JCT，也可以用$代替。

API

$等同于JCT。当页面中引用jquery时，使用JCT。

DOM选择器：$(selector)

参数：
selector => string

传入一个css选择器字符串，得到一个JCT对象，通过JCT对象能够对dom进行各种操作。

示例1-1

var $dom = $("body");
alert($dom.length); // 1
alert($dom[0].tagName); // "BODY"
